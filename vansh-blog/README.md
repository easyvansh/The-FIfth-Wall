# Vansh Singh — Blog (V1)

This repository contains a Next.js + Tailwind starter for Vansh Singh's personal blog following the AGENTS.md spec.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Create and run Sanity Studio (see /studio/README.md)

4. Deploy to Vercel and set environment variables described in /studio/README.md
